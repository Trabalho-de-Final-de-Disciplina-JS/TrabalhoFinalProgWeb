# Projeto Final Programação Web, Pacote de Softwares
![Visual Studio Code](https://img.shields.io/badge/Visual%20Studio%20Code-0078d7.svg?style=for-the-badge&logo=visual-studio-code&logoColor=white)

## Descrição:
Este foi o último projeto que a nossa turma recebeu na disciplina de Programação Web, cada um de nós, Eduarda, Leon e Arthur trabalhamos em partes neste projeto dividindo tarefas. O site consiste em um pacote de sistemas para formar calculos de conversão de medida tais como comprimento, volume, temperatura, peso e um sistema que gera uma mensagem de motivação aleatória.
